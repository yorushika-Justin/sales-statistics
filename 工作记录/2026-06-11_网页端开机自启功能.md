# 网页端开机自启功能开发记录

> 日期：2026-06-11
> 版本：网页端 v1.5
> 变更类型：功能新增

## 一、需求背景

用户希望销售汇总网页端服务能够开机自动启动，配合 Edge 浏览器中保存的 `http://127.0.0.1:5000` 便签页实现一键使用，无需每次手动双击 `启动网页端.vbs`。

**核心要求**：
- 开机自动启动 Flask 服务
- 后台内存占用低（空闲时约 55MB，CPU 接近 0%）
- 不影响其他应用
- exe 端方案不受影响
- 所有修改限制在 `网页端` 文件夹内
- 开机自启失败时，原有 vbs/bat 仍能正常使用

## 二、技术方案评估

### 2.1 方案对比

| 方案 | 难度 | 可控性 | 推荐度 |
|---|---|---|---|
| Windows 任务计划程序 | 低 | 高（可延迟、可条件触发） | **推荐** |
| 注册表启动项 | 低 | 中（立即启动，无延迟） | 备选 |
| 启动文件夹快捷方式 | 最低 | 低（用户可能误删） | 不推荐 |

### 2.2 最终选择：任务计划程序

**理由**：
1. 可设置延迟启动（登录后 30 秒），避免拖慢开机速度
2. 不需要管理员权限即可运行任务（`/rl limited`）
3. 支持查看状态、注销等管理操作
4. 系统级管理，比注册表/启动文件夹更可靠

### 2.3 内存占用分析

| 状态 | 内存占用 | CPU |
|---|---|---|
| 空闲（等待请求） | ~55 MB | 接近 0% |
| 处理 xls/xlsx 文件时 | ~60-80 MB（短暂峰值） | 短暂升高 |

**结论**：空闲时几乎不消耗 CPU，不会影响其他应用。

## 三、实现细节

### 3.1 新增文件

**`网页端/开机自启.bat`** — 管理脚本

功能：
- `[1] 注册开机自启`：创建任务计划程序任务 + 生成静默 vbs
- `[2] 注销开机自启`：删除任务计划程序任务 + 清理静默 vbs
- `[3] 查看当前状态`：查询任务计划程序任务状态

关键命令：
```bat
:: 注册任务（登录后延迟 30 秒启动，受限用户权限）
schtasks /create /tn "销售汇总网页端" /tr "wscript.exe \"启动网页端_静默.vbs\"" /sc onlogon /delay 0000:30 /f /rl limited

:: 注销任务
schtasks /delete /tn "销售汇总网页端" /f

:: 查看状态
schtasks /query /tn "销售汇总网页端" /fo list
```

**`网页端/启动网页端_静默.vbs`** — 自动生成

由 `开机自启.bat` 注册时自动生成，内容：
```vbs
Set WshShell = CreateObject("WScript.Shell")
WshShell.Run "pythonw ""D:\Code\project10-销售统计\网页端\web_server.py"" --silent", 0, False
```

### 3.2 修改文件

**`网页端/web_server.py`** — 新增 `--silent` 参数支持

修改 `if __name__ == '__main__'` 块：

```python
if __name__ == '__main__':
    # 检查是否为静默模式（开机自启时使用 --silent 参数）
    silent_mode = '--silent' in sys.argv

    # 启动检查（静默场景下必须用弹窗，不能用 print）
    if not silent_mode:
        _check_already_running()  # 端口已被占 → 弹窗询问 + sys.exit(0)
    else:
        # 静默模式：端口已占则静默退出，不弹窗
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(0.3)
        try:
            sock.connect(('127.0.0.1', PORT))
            sock.close()
            sys.exit(0)  # 服务已在运行，静默退出
        except (ConnectionRefusedError, socket.timeout, OSError):
            pass
        finally:
            sock.close()

    _check_output_dir()       # 输出目录不存在 → 弹窗报错 + sys.exit(1)

    # 在新线程中打开浏览器（非静默模式才打开）
    if not silent_mode:
        threading.Thread(target=open_browser, daemon=True).start()

    # 启动 Flask 服务（threaded=True 支持多标签页并发）
    try:
        app.run(host='127.0.0.1', port=PORT, debug=False, threaded=True)
    except OSError as e:
        if 'address already in use' in str(e) or '通常每个套接字地址' in str(e):
            if not silent_mode:
                _show_error_popup(...)
        else:
            if not silent_mode:
                _show_error_popup(...)
            raise
```

**静默模式行为**：
- 端口已占 → 静默退出（不弹窗）
- 端口未占 → 启动服务（不弹浏览器）
- 输出目录不存在 → 弹窗报错（仍需提示，否则服务无法工作）

### 3.3 向后兼容

## 四、使用说明

### 4.1 注册开机自启

1. 双击 `网页端/开机自启.bat`
2. 输入 `1` 回车
3. 完成后 Edge 浏览器中保存的 `http://127.0.0.1:5000` 便签页可直接使用

### 4.2 注销开机自启

1. 双击 `网页端/开机自启.bat`
2. 输入 `2` 回车

### 4.3 手动启动（原有方式）

- 双击 `启动网页端.vbs`（静默启动 + 弹浏览器）
- 双击 `启动网页端.bat`（cmd 窗口 + 弹浏览器）

### 4.4 停止服务

- 网页底部"停止服务"按钮
- 任务管理器结束 `pythonw.exe`

## 六、注意事项

1. **权限**：注册/注销任务需要管理员权限（脚本会自动请求 UAC 提权）
2. **Python 路径**：当前硬编码为 `D:\Python\pythonw.exe`，如重装 Python 需重新注册
3. **端口占用**：静默模式下端口已占会静默退出，不会弹窗干扰用户
4. **内存占用**：空闲时约 55MB，CPU 接近 0%，不影响其他应用
5. **停止服务**：网页底部按钮 / 任务管理器均可停止
