# 网页端-2 工作记录

> 日期：2026-06-18
> 版本：v2.0
> 变更类型：重构 + 功能不变

---

## 一、需求背景

用户需要为销售汇总网页端创建独立备份（网页端-2），以便在更换电脑后使用。核心需求：

1. 功能与原网页端完全一致
2. 输出路径改为可配置（不再硬编码）
3. 所有文件独立，不依赖原网页端目录
4. 同时生成新的 exe 文件

**约束条件**：
- 新电脑的输出路径、Python 路径、用户名可能变化
- 需要等用户确认最终路径后再完成配置
- 技术文档、工作记录等都要是新的

---

## 二、技术方案

### 2.1 路径可配置方案

**选择：配置文件方案**

- 新增 `config.json` 存储输出路径、字体路径、端口
- `web_server.py` 启动时读取配置
- 优点：改路径不改代码，用户改一个文件即可

### 2.2 文件结构

```
网页端-2/
├── web_server.py          ← 主程序（从 config.json 读取配置）
├── config.json            ← 配置文件
├── 启动网页端.vbs          ← 静默启动器
├── 启动网页端.bat          ← 调试启动器
├── 开机自启.bat            ← 开机自启管理脚本
├── index.html             ← 独立 HTML 副本
├── 销售汇总.exe            ← 打包后的 exe
├── 技术文档.md            ← 技术文档
└── 工作记录.md            ← 本文件
```

## 三、实现细节

### 3.1 配置文件 (`config.json`)

```json
{
    "output_dir": "D:\\16678\\相册\\销售额",
    "font_path": "C:\\Windows\\Fonts\\msyh.ttc",
    "port": 5000
}
```

**说明**：
- `output_dir`：PNG 输出目录（占位，等用户提供最终路径）
- `font_path`：字体文件路径（占位）
- `port`：Flask 服务端口

### 3.2 主程序 (`web_server.py`)

**主要修改**：
- 文件头版本号改为 v2.0
- 启动时读取 `config.json` 获取 `OUTPUT_DIR`、`FONT_PATH`、`PORT`
- 其他功能完全不变

**关键代码**：
```python
# 读取配置文件
CONFIG_PATH = os.path.join(os.path.dirname(os.path.abspath(__file__)), 'config.json')
try:
    with open(CONFIG_PATH, 'r', encoding='utf-8') as f:
        config = json.load(f)
    OUTPUT_DIR = config.get('output_dir', r'D:\16678\相册\销售额')
    FONT_PATH = config.get('font_path', r'C:\Windows\Fonts\msyh.ttc')
    PORT = config.get('port', 5000)
except Exception as e:
    print(f'[警告] 读取配置文件失败: {e}，使用默认值')
    OUTPUT_DIR = r'D:\16678\相册\销售额'
    FONT_PATH = r'C:\Windows\Fonts\msyh.ttc'
    PORT = 5000
```

### 3.3 启动器

| 文件 | 内容 | 说明 |
|---|---|---|
| `启动网页端.vbs` | `CreateObject("Wscript.Shell").Run "pythonw ""web_server.py""", 0, False` | 静默启动 + 弹浏览器 |
| `启动网页端.bat` | `python web_server.py` + `pause` | cmd 窗口 + 弹浏览器 |

### 3.4 开机自启脚本

**改进**：
- 界面改为中文（原版是英文）
- 逻辑不变：注册/注销/查看状态
- 使用 Windows 任务计划程序

**关键命令**：
```bat
:: 注册任务（登录后延迟 30 秒启动，受限用户权限）
schtasks /create /tn "销售汇总网页端" /tr "wscript.exe \"启动网页端_静默.vbs\"" /sc onlogon /delay 0000:30 /f /rl limited

:: 注销任务
schtasks /delete /tn "销售汇总网页端" /f

:: 查看状态
schtasks /query /tn "销售汇总网页端" /fo list
```

### 3.5 独立 HTML (`index.html`)

- 从原网页端的 `index.html` 复制
- 更新支持 `.xlsx` 格式（原版只支持 `.xls`）
- 供 Edge 浏览器直接打开（离线可用）

## 五、新电脑部署环境要求

将网页端-2 部署到新电脑时，需要准备以下环境：

### 5.1 Python 环境

- **Python 版本**：3.x（推荐 3.10+，当前开发环境 3.13.12）
- **安装路径**：自定义（如 `D:\Python\`），需记录路径后续配置用
- **pip**：随 Python 自带

### 5.2 Python 依赖包

```powershell
pip install Flask==3.1.3 xlrd==1.2.0 openpyxl==3.1.5 Pillow==12.1.1
```

| 包 | 版本 | 用途 | 备注 |
|---|---|---|---|
| Flask | 3.1.3 | Web 框架 | |
| xlrd | 1.2.0 | 读 .xls | **注意：2.0+ 不支持 .xls，必须装 1.2.0** |
| openpyxl | 3.1.5 | 读 .xlsx | |
| Pillow | 12.1.1 | 绘图生成 PNG | |

### 5.3 系统要求

- **操作系统**：Windows（.vbs 启动器、tkinter、任务计划程序均为 Windows 特有）
- **字体**：微软雅黑 `msyh.ttc`（Windows 自带，路径 `C:\Windows\Fonts\msyh.ttc`）
- **浏览器**：Edge（推荐）或其他

### 5.4 配置文件修改

部署后需修改以下配置：

**1. `config.json`**（输出路径）
```json
{
    "output_dir": "新电脑的实际输出路径",
    "font_path": "C:\\Windows\\Fonts\\msyh.ttc",
    "port": 5000
}
```

**2. `开机自启.bat`**（Python 路径，第 10 行）
```bat
set "PYTHONW_PATH=新电脑的Python安装路径\pythonw.exe"
```

### 5.5 输出目录

- `config.json` 中配置的 `output_dir` 目录需**预先创建**
- 程序不会自动创建该目录，目录不存在时会报错退出
