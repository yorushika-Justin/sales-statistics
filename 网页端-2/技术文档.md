## 1. 项目概述

### 1.1 目标
将供应商日销售汇总 xls/xlsx 文件转换为 PNG 表格图片。通过本地 Web 服务实现网页拖拽操作，PNG 保存到固定目录并自动打开。

### 1.2 使用场景
- 用户双击 `启动网页端.vbs` 静默启动 Flask 服务（无黑窗口）
- 浏览器自动打开 `http://127.0.0.1:5000`
- 拖入 xls/xlsx 文件 → 后端处理 → 生成 PNG → 系统默认图片查看器打开
- 服务常驻（约 55MB），可一直使用；想停可在网页底部点"停止服务"

### 1.3 与原网页端的区别

| 特性 | 原网页端 | 网页端-2 |
|---|---|---|
| 输出路径 | 硬编码在 `汇总脚本.py` | 从 `config.json` 读取 |
| 配置方式 | 改代码 | 改 `config.json` |
| 开机自启脚本 | 英文界面 | 中文界面 |
| 依赖 | 共用 `汇总脚本.py` | 共用 `汇总脚本.py` |

### 1.4 启动方式

| 启动方式 | 适用场景 | 表现 |
|---|---|---|
| **双击 `启动网页端.vbs`** | 日常使用（推荐） | 静默启动，无窗口，浏览器自动打开 |
| **双击 `启动网页端.bat`** | 开发/调试 | 弹 cmd 窗口，看得见 Flask 日志 |
| `python web_server.py` | 命令行 | 同 .bat |
| `python web_server.py --silent` | 开机自启 | 静默启动，不弹浏览器 |

## 5. 程序结构 (`web_server.py`)

### 5.1 主要组件

| 组件 | 职责 |
|---|---|
| `HTML_PAGE` | 内嵌前端页面（纯 HTML/JS/CSS） |
| `@app.route('/')` | 返回前端页面 |
| `@app.route('/upload', methods=['POST'])` | 接收 xls/xlsx，调用核心函数，返回 JSON 结果 |
| `@app.route('/shutdown', methods=['POST'])` | 停止服务（调 `os._exit(0)`） |
| `_show_popup(msg, kind)` | 统一的 tkinter 弹窗工具函数 |
| `_check_already_running()` | 检测端口是否被占（防重复双击） |
| `_check_output_dir()` | 检测 OUTPUT_DIR 是否存在 |
| `_show_error_popup(msg)` | tkinter 弹窗（静默场景下提示错误） |
| `open_browser()` | socket 轮询探测端口就绪后打开浏览器 |
| `if __name__ == '__main__'` | 启动检查 → 启动 Flask（threaded=True） |

### 5.2 核心函数（复用自 汇总脚本.py）

| 函数 | 来源 | 职责 |
|---|---|---|
| `read_xls(path, log_path)` | 汇总脚本.py | 读 xls，按品牌汇总，返回 `(data, date_str)` |
| `read_xlsx(path, log_path)` | 汇总脚本.py | 读 xlsx，按品牌汇总，返回 `(data, date_str)` |
| `read_xls_items(path, log_path)` | 汇总脚本.py | 读 xls，按品名逐行提取明细 |
| `read_xlsx_items(path, log_path)` | 汇总脚本.py | 读 xlsx，按品名逐行提取明细 |
| `draw_table(data, date_str, output_path)` | 汇总脚本.py | Pillow 画品牌汇总表格 PNG |
| `draw_table_detail(items, date_str, output_path)` | 汇总脚本.py | Pillow 画品名明细表格 PNG |

### 5.3 启动检查流程

```
web_server.py 被执行
    │
    ▼
[0] 读取 config.json
    │   获取 OUTPUT_DIR, FONT_PATH, PORT
    │
    ▼
[1] _check_already_running()
    │   socket 探测 127.0.0.1:PORT
    │
    ├─ 已占用 → _show_popup("服务已运行，是否打开浏览器？", kind='yesno')
    │          ├─ 是 → os.startfile(http://127.0.0.1:PORT)
    │          └─ 否 → sys.exit(0)
    │
    └─ 未占用 → 继续
    │
    ▼
[2] _check_output_dir()
    │   os.path.isdir(OUTPUT_DIR)
    │
    ├─ 不存在 → _show_popup("输出目录不存在: ...")
    │           sys.exit(1)
    │
    └─ 存在 → 继续
    │
    ▼
[3] 启动 Flask (主线程)
    │   app.run(host='127.0.0.1', port=PORT, debug=False, threaded=True)
    │
    ▼
[4] open_browser() (后台线程)
    │   socket 轮询探测 PORT 端口就绪（最多 5 秒）
    │   └─ 就绪 → os.startfile(http://127.0.0.1:PORT)
```

### 6.1 新电脑部署环境要求

将网页端-2 部署到新电脑时，需要准备以下环境：

**一、Python 环境**

| 项目 | 要求 | 说明 |
|---|---|---|
| Python 版本 | 3.x（推荐 3.10+） | 当前开发环境为 3.13.12 |
| 安装路径 | 自定义（如 `D:\Python\`） | 需记录路径，后续配置用 |
| pip | 随 Python 自带 | 用于安装依赖包 |

**二、Python 依赖包安装**

```powershell
pip install Flask==3.1.3 xlrd==1.2.0 openpyxl==3.1.5 Pillow==12.1.1
```

| 包 | 版本 | 用途 |
|---|---|---|
| Flask | 3.1.3 | Web 框架 |
| xlrd | 1.2.0 | 读 .xls（注意：2.0+ 不支持 .xls） |
| openpyxl | 3.1.5 | 读 .xlsx |
| Pillow | 12.1.1 | 绘图生成 PNG |

**三、系统要求**

| 项目 | 要求 | 说明 |
|---|---|---|
| 操作系统 | Windows | .vbs 启动器、tkinter、任务计划程序均为 Windows 特有 |
| 字体 | 微软雅黑 `msyh.ttc` | Windows 自带，路径 `C:\Windows\Fonts\msyh.ttc` |
| 浏览器 | Edge（推荐）或其他 | 用于访问网页端 |

**四、配置文件修改**

部署后需修改以下配置：

1. **`config.json`**（输出路径）
   ```json
   {
       "output_dir": "新电脑的实际输出路径",
       "font_path": "C:\\Windows\\Fonts\\msyh.ttc",
       "port": 5000
   }
   ```

2. **`开机自启.bat`**（Python 路径，第 10 行）
   ```bat
   set "PYTHONW_PATH=新电脑的Python安装路径\pythonw.exe"
   ```

**五、输出目录**

- `config.json` 中配置的 `output_dir` 目录需**预先创建**
- 程序不会自动创建该目录，目录不存在时会报错退出

## 8. 开机自启

### 8.1 功能说明

支持开机自动启动 Flask 服务，配合 Edge 浏览器便签页实现一键使用。

**特点**：
- 使用 Windows 任务计划程序（非注册表/启动文件夹，更可靠）
- 登录后延迟 30 秒启动（避免拖慢开机速度）
- 静默模式（不自动弹浏览器，适合后台常驻）
- 端口已占用时静默退出（避免重复启动报错）

### 8.2 使用方法

**注册开机自启**：
1. 右键 `网页端-2/开机自启.bat` → 以管理员身份运行
2. 选择 `[1] 注册开机自启`
3. 完成后 Edge 浏览器中保存的 `http://127.0.0.1:5000` 便签页可直接使用

**注销开机自启**：
1. 右键 `网页端-2/开机自启.bat` → 以管理员身份运行
2. 选择 `[2] 注销开机自启`

**查看当前状态**：
1. 双击 `网页端-2/开机自启.bat`
2. 选择 `[3] 查看当前状态`

### 8.3 文件说明

| 文件 | 用途 |
|---|---|
| `开机自启.bat` | 管理脚本（注册/注销/查看状态） |
| `启动网页端_静默.vbs` | 自动生成，任务计划程序调用（带 `--silent` 参数） |
| `启动网页端.vbs` | 原有手动启动器（不受影响，仍会弹浏览器） |
| `启动网页端.bat` | 原有调试启动器（不受影响） |

## 9. 输出文件

- 路径：`config.json` 中的 `output_dir` + `<日期>\`（按日期自动创建子文件夹）
- 命名：`YYYY-MM-DD.png`（品牌汇总）+ `YYYY-MM-DD_品名.png`（品名明细）
- 日期从 xls/xlsx 内部 Row 2 提取，结束日期 -1 天
- 同名覆盖无提示

## 10. 已知限制

| 限制 | 影响 | 状态 |
|---|---|---|
| 服务常驻约 55MB 内存 | 后台长期运行 | 设计如此（用户已确认可接受） |
| 端口 5000 固定 | 被占用时启动失败 | 可通过 config.json 修改 |
| 仅本机访问 | 不支持远程 | 设计如此（安全） |
| 依赖 Python 环境 | 用户需安装 Python + Flask | 可打包成 exe |
| .vbs 启动器 | 仅 Windows 支持 | 设计如此（本项目仅 Windows） |
