## 1. 项目概述

### 1.1 目标
将供应商日销售汇总 xls 文件转换为 PNG 表格图片。通过本地 Web 服务实现网页拖拽操作，PNG 保存到固定目录并自动打开。

### 1.2 使用场景
- 用户双击 `启动网页端.vbs` 静默启动 Flask 服务（无黑窗口）
- 浏览器自动打开 `http://127.0.0.1:5000`
- 拖入 xls 文件 → 后端处理 → 生成 PNG → 系统默认图片查看器打开
- 服务常驻（约 55MB），可一直使用；想停可在网页底部点"停止服务"

### 1.3 与 exe 方案的对比

| 特性 | exe 方案 | 网页端方案 |
|---|---|---|
| 启动方式 | 双击 exe | 双击 .vbs 启动器（静默） |
| 交互方式 | 拖拽到图标 / GUI 窗口 | 浏览器网页拖拽 |
| 多任务 | 需多次拖拽 | Edge 多标签页并行 |
| 打包 | pyinstaller（34 MB） | 无需打包（.py + .vbs） |
| 依赖 | 全部打包进 exe | 需 Python 环境 + Flask |
| 停服务 | 关 GUI 窗口 | 网页底部按钮 / 任务管理器 |

### 1.4 启动方式（v1.1 新增）

| 启动方式 | 适用场景 | 表现 |
|---|---|---|
| **双击 `启动网页端.vbs`** | 日常使用（推荐） | 静默启动，无窗口，浏览器自动打开 |
| **双击 `启动网页端.bat`** | 开发/调试 | 弹 cmd 窗口，看得见 Flask 日志 |
| `python web_server.py` | 命令行 | 同 .bat |

## 4. 程序结构 (`web_server.py`)

### 4.1 主要组件

| 组件 | 职责 |
|---|---|
| `HTML_PAGE` | 内嵌前端页面（纯 HTML/JS/CSS） |
| `@app.route('/')` | 返回前端页面 |
| `@app.route('/upload', methods=['POST'])` | 接收 xls，调用核心函数，返回 JSON 结果 |
| `@app.route('/shutdown', methods=['POST'])` | 停止服务（调 `os._exit(0)`） |
| `_show_popup(msg, kind)` | **v1.2 新增** 统一的 tkinter 弹窗工具函数 |
| `_check_already_running()` | 检测端口是否被占（防重复双击），复用 `_show_popup` |
| `_check_output_dir()` | 检测 OUTPUT_DIR 是否存在，复用 `_show_popup` |
| `_show_error_popup(msg)` | tkinter 弹窗（静默场景下提示错误），复用 `_show_popup` |
| `open_browser()` | **v1.2 改进** socket 轮询探测端口就绪后打开浏览器 |
| `if __name__ == '__main__'` | 启动检查 → 启动 Flask（threaded=True） |

### 4.2 核心函数（复用自 汇总脚本.py）

| 函数 | 来源 | 职责 |
|---|---|---|
| `read_xls(path, log_path)` | 汇总脚本.py | 读 xls，按品牌汇总，返回 `(data, date_str)` |
| `read_xls_items(path, log_path)` | 汇总脚本.py | **v1.3 新增** 读 xls，按品名逐行提取明细，返回 `[(品名, 品牌, 条码, 数量, 金额, 库存), ...]` |
| `draw_table(data, date_str, output_path)` | 汇总脚本.py | Pillow 画品牌汇总表格 PNG（2 列） |
| `draw_table_detail(items, date_str, output_path)` | 汇总脚本.py | **v1.3 新增** Pillow 画品名明细表格 PNG（7 列：行号/国际条码/品名/品牌/销售数量/销售金额/库存） |
| `OUTPUT_DIR` | 汇总脚本.py | 固定输出目录 `D:\16678\相册\销售额` |

### 4.3 启动检查流程

```
web_server.py 被执行
    │
    ▼
[1] _check_already_running()
    │   socket 探测 127.0.0.1:PORT
    │
    ├─ 已占用 → _show_popup("服务已运行，是否打开浏览器？", kind='yesno')
    │          ├─ 是 → os.startfile(http://127.0.0.1:PORT)
    │          └─ 否 → sys.exit(0)
    │
    └─ 未占用 → 继续
    │
    ▼
[2] _check_output_dir()
    │   os.path.isdir(OUTPUT_DIR)
    │
    ├─ 不存在 → _show_popup("输出目录不存在: ...")
    │           sys.exit(1)
    │
    └─ 存在 → 继续
    │
    ▼
[3] 启动 Flask (主线程)
    │   app.run(host='127.0.0.1', port=PORT, debug=False, threaded=True)
    │
    ▼
[4] open_browser() (后台线程)
    │   socket 轮询探测 PORT 端口就绪（最多 5 秒）
    │   └─ 就绪 → os.startfile(http://127.0.0.1:PORT)
```

### 4.4 错误处理（v1.1 更新）

| 错误场景 | 提示方式 | 处理位置 |
|---|---|---|
| 端口 5000 被其他程序占 | tkinter 弹窗 | `_check_already_running()` 内的 socket 探测 |
| 服务已运行（用户重复双击） | tkinter 弹窗询问"是否打开浏览器" | `_check_already_running()` |
| 输出目录不存在 | tkinter 弹窗 | `_check_output_dir()` |
| Flask 启动失败（其他 OSError） | tkinter 弹窗 + 写日志 | `if __name__` 块的 try/except |
| 运行时 /upload 错误 | 网页端 JSON 返回 | `/upload` 路由（已有） |
| 用户点"停止服务"按钮 | 二次确认 → `os._exit(0)` | `/shutdown` 路由 |

## 9. 开机自启（v1.5 新增）

### 9.1 功能说明

支持开机自动启动 Flask 服务，配合 Edge 浏览器便签页实现一键使用。

**特点**：
- 使用 Windows 任务计划程序（非注册表/启动文件夹，更可靠）
- 登录后延迟 30 秒启动（避免拖慢开机速度）
- 静默模式（不自动弹浏览器，适合后台常驻）
- 端口已占用时静默退出（避免重复启动报错）

### 9.2 使用方法

**注册开机自启**：
1. 双击 `网页端/开机自启.bat`
2. 选择 `[1] 注册开机自启`
3. 完成后 Edge 浏览器中保存的 `http://127.0.0.1:5000` 便签页可直接使用

**注销开机自启**：
1. 双击 `网页端/开机自启.bat`
2. 选择 `[2] 注销开机自启`

**查看当前状态**：
1. 双击 `网页端/开机自启.bat`
2. 选择 `[3] 查看当前状态`

### 9.3 文件说明

| 文件 | 用途 |
|---|---|
| `开机自启.bat` | 管理脚本（注册/注销/查看状态） |
| `启动网页端_静默.vbs` | 自动生成，任务计划程序调用（带 `--silent` 参数） |
| `启动网页端.vbs` | 原有手动启动器（不受影响，仍会弹浏览器） |
| `启动网页端.bat` | 原有调试启动器（不受影响） |

### 9.4 技术实现

- **任务计划程序**：`schtasks /create /tn "销售汇总网页端" /tr "wscript.exe \"启动网页端_静默.vbs\"" /sc onlogon /delay 0000:30 /f /rl limited`
- **静默模式**：`web_server.py` 支持 `--silent` 参数，启动时不弹浏览器，端口已占时静默退出
- **向后兼容**：原有的 `启动网页端.vbs` / `启动网页端.bat` 行为不变

### 9.5 注意事项

- 注册/注销需要**管理员权限**（右键 `开机自启.bat` → 以管理员身份运行）
- 如果 Python 路径变更（如重装），需重新注册
- 内存占用：空闲时约 55MB，CPU 接近 0%
- 停止服务：网页底部"停止服务"按钮 / 任务管理器结束 `pythonw.exe`
- 注册成功后会生成 `启动网页端_静默.vbs`（任务计划程序调用，勿手动删除）
